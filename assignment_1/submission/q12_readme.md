## Problem Statement

- Similar to Q04 but with streaming interface. Find<br>
all the unique prime numbers from given input<br>
using streaming interface<br>
- <strong>Input</strong><br>
  – i/p 1: number of mapped instances<br>
  – i/p 2: streaming input of 1M+ positive integers<br>
  • Numbers may have duplicate values<br>
- <strong>Output</strong><br>
  – o/p: unique list of all prime numbers<br>

$ bash local_demo.sh
